# Simple EMR

Simple electronic medical records (EMR) as a school assignment

## 구현 내용 요구사항

1. 환자 10명의 정보를 저장 및 수정
   * 이름, 나이, 성별, 주소
2. 각 환자당 10개의 환자 차트 저장 및 수정
   * 내원일, 호소증상, 환자평가, 진단, 치료 및 계획
3. 이외에 독창적인 내용

## 개발 환경 요구사항

* 언어: Python3
* GUI: Tkinter
